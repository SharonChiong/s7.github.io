Python 字符编码

Python 字符串

官方文档中介绍 Python 3 字符串：Strings are immutable sequences of Unicode code points.翻译过来就是，字符串是由 Unicode 码点组成的不可变序列。

字符串在 Python 2 和 Python 3 的不同

在 Python 2 中，字符串有两种类型：str 和 unicode。str 对象存储的是 bytes(字节码)，有 u 前缀的是 unicode 对象，存储的是 code points(码点)。注意，这里的 unicode 才是真正意义上的字符串。

字符串 是一种数据结构。数据结构 = 源数据 + 数据描述，数据描述 指的是数据使用的字符编码，比如 Unicode 字符集，ASCII 码等。而字节码没有数据描述。

字节 是计算机中存储数据的单元，字符 是人们使用的记号，如 ‘1’, ‘a’, ‘&’, …

在 Python 2 中，字符串使用的默认编码格式是 ASCII 编码，Python 3 默认支持 Unicode 字符集。

Python 3 的字符串也有两种类型：bytes 和 str，严格区分文本和二进制数据。bytes 相当于 Python 2 的 str，str 相当于 Python 2 的 unicode。这样一来，str 成为了真正意义上的字符串。Python 3 字符串支持多语言。

Python 3 字符串类型在内存中以 Unicode 表示，但在网络上传输或保存到磁盘里，需要转换成 bytes 字节流。

bytes 和 unicode 的转换通过 编码解码(encode & decode) 的方式。

Unicode 和 UTF-8

还记得 Python 3 中字符串的定义吗？Unicode 码点就是用 Unicode 编码的字符。那 Unicode 是什么？

Unicode 是一种字符集，不同于 ASCII 码 只规定了 128 个字符编码，Unicode 可以容纳的规模已经超过 100 万。

但是 Unicode 只规定了符号的二进制代码，没有规定这个二进制代码如何存储，为了解决存储问题和优化存储空间，对 Unicode 进一步转换成 UTF-8 编码。UTF-8 是一种变长的编码方式。比如，英文字符只需要一个字节存储，中文需要三个字节。

字符编码与解码

  • 编码：将字符串编码成 bytes 对象；

  • 解码：将 bytes 对象解码成字符串。

  • 参数：

    1. encoding 默认编码为 utf-8;

    2. errors 设置不同错误的处理方案，默认为 strict，参数其他值还有 ignore, replace, xmlcharrefreplace, backslashreplace等。ignore 表示忽略非法字符，replace 表示会用 ? 取代非法字符，xmlcharrefreplace 表示使用 XML 的字符引用。

本章小结

  • 区分 Python 2 和 Python 3 中的字符串;

  • 认识 Unicode 和 UTF-8;

  • 在 Python 中，使用 encode() 编码 Unicode 和 使用 decode() 解码 bytes.
