时间复杂度

基本操作执行次数

设 T(n) 为程序基本操作执行次数的函数，n 为输入规模，以下4个场景对应程序中最常见的4中执行方式👇

  1. T(n) = 3n, 执行次数为线性的；
  2. T(n) = 5logn, 执行次数是用对数计算的；
  3. T(n) = 2, 执行次数是常量；
  4. T(n) = 0.5n^2+0.5n, 执行次数是用多项式计算的。

渐进时间复杂度

asymptotic time complexity, 如存在函数 f(n), 使得当 n 趋近于无穷大时，T(n)/f(n) 的极限值为不等于零的常量，则称 f(n) 是 T(n) 的同数量级函数。记作 T(n) = O(f(n)), 称为 O(f(n)), O 为算法的渐进时间复杂度，简称 时间复杂度。

渐进时间复杂度用大写 O 表示，也称为 大O表示法。

如何推导出时间复杂度？

  1. 如果运行时间是常数量级，则用常数1表示；
  2. 只保留时间函数中的最高阶项；
  3. 如果最高阶项存在，则省去最高阶项前面的系数。

👆上面的4个场景分别对应 T(n)=O(n), T(n)=O(logn), T(n)=O(1), T(n)=O(n^2).

当 n 取值足够大时，O(1)<O(logn)<O(n)<O(n^2).

还有其他的时间复杂度，O(nlogn), O(n^3), O(mn), O(2^n), O(n!).

空间复杂度

space complexity, 对一个算法在运行过程中临时占用存储空间大小的量度，同样使用大O表达式。

S(n)=O(f(n)), 其中 n 为问题的规模，f(n) 为算法所占存储空间的函数。

常见的空间复杂度：

  1. 常量空间：O(1)
  2. 线性空间；O(n)
  3. 二维空间；O(n^2)
  4. 递归空间：O(n), 方法调用栈 包括 进栈 和 出栈。
