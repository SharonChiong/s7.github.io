Python 日期和时间的基本使用(1)

官方文档和源码

  • docs: datetime - Basic date and time types

  • Source code: Lib/datetime.py

Available Types(可用类型)

  • datetime.date(日期) – Attributes(属性): year, month, day.

  • datetime.time(时间) – Attributes: hour, minute, second, microsecond, tzinfo.

  • datetime.datetime(日期和时间的组合) – Attributes: year, month, day, hour, minute, second, microsecond, tzinfo.

  • datetime.timedelta(两个 date, time, 或 datetime 的时间差)

  • datetime.tzinfo(时区信息对象的抽象基类)

  • datetime.timezone(将 tzinfo 抽象基类实现为与 UTC 的固定偏移量的类)

timedelta 对象

  • 实例的属性(只读): days, seconds, microseconds

  • 支持 + - * / // 等数值操作
